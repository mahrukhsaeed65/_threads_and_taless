# 🧶 Threads and Tales — Crochet Store Website

A simple, elegant, and free-to-deploy website for my crochet small business **Threads and Tales**.

---

## 🌸 Project Goal

Create a minimal and aesthetic website to:

* Showcase crochet products
* Match brand colors and theme
* Allow customers to place orders easily
* Deploy for free using modern tools

---

## 🛠 Tech Stack

* Windsurf (Development)
* GitHub (Version Control)
* Vercel (Deployment)
* HTML + CSS + JavaScript (Frontend)

---

## 📁 Project Structure

```
threads-and-tales/
```

## 🎨 Design Guidelines

* Use soft, cozy colors (based on logo)
* Recommended palette:

  * Cream / Beige (#f5f0ea)
  * Soft Pink (#e8b4b8)
  * Brown (#6b4f4f)
* Clean fonts (e.g., Poppins, Playfair Display)

## 📦 Adding Products

Inside `index.html`, add product cards like:

```html
<div class="product">
  <img src="assets/images/bag1.jpg" alt="Crochet Bag">
  <h3>Rose Crochet Bag</h3>
  <p>Rs. 1200</p>
  <a href="https://wa.me/92XXXXXXXXXX?text=I want to order Rose Crochet Bag" target="_blank">
    Order on WhatsApp
  </a>
</div>
```

---

## 📲 WhatsApp Order System (No Backend Needed)

Replace with your number:

```
https://wa.me/92XXXXXXXXXX
```

Example message:

```
I want to order [Product Name]
```

---

## 📧 Email Option (Optional)

```html
<a href="mailto:your@email.com">Order via Email</a>
```

---

## 🚀 Deployment (Vercel - Free)

1. Push project to GitHub
2. Go to Vercel
3. Click "Add New Project"
4. Import your GitHub repo
5. Click Deploy

Done 🎉
